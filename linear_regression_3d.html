Key Formulas (correct)

SST = Σ(yi − ȳ)²        (total variation)
SSR = Σ(ŷi − ȳ)²        (explained / regression)
SSE = Σ(yi − ŷi)²        (residual / unexplained)
SST = SSR + SSE

R² = SSR / SST          (0 .. 1)
MSE = SSE / n
RMSE = √MSE
R = corr(y, ŷ) ≈ sign(slope)·√R² (simple linear)
